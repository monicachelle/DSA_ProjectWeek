# Smart Parking Management System

This project demonstrates three simple Java programs for a college-level DSA mini project.

## Project Modules

- `CO1_AVL_VehicleManagement`: Vehicle record storage using an AVL tree structure.
- `CO2_SegmentTree_TrafficAnalysis`: Parking traffic analysis using a segment tree for range sums and updates.
- `CO3_Graph_RouteOptimization`: Parking route navigation using graph traversal (BFS and DFS).

Each Java file is independent and uses basic Java features suitable for beginners.
